# Encora Practical

APIs included
SignUp,
Authentication and Authorization,
CRUD of Note by a user

## Prerequisite (Development)

| Module | Version |
| --- | --- |
| Node | 12.16.0 |
| Npm | 6.13.0 |
| MySQL | 5.6 or 5.7 |



## Running Project In Local
``` bash
$ git clone -b <branchName> <gitUrl> <projectName>

$ vi .env.dev (need to change configuration of DB)

$ npm install

$ npm start

```
